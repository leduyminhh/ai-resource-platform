# AI Resource Platform Specification (ARPS)

A vendor-neutral, resource-oriented specification repository for building reusable AI development platforms.

## Core Idea

Everything is a canonical Resource.

Plugin, Skill, Prompt, Workflow, Policy, Adapter, Package and Registry are all Resource kinds. Runtime engines operate on resources and graphs, not vendor-specific IDE models.

## Structure

- `docs/`: RFC specifications
- `schemas/`: canonical schemas
- `examples/`: example manifests
- `reference/`: reference notes
- `tools/`: placeholder tooling
